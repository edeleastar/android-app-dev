Event-Handling Lab

This is a short lab whose aim is to provide a detailed explanation of event handling

  • A small project is developed to show how the the process works.
  • Three separate approaches are demonstrated:
    • A standalone method
    • A method that uses an anonymous class
    • A method we refer to as the delegate method
  • The key classes used in this lab have been designed to correspond closely to those used in MyRent.
  • Before working on this lab it would help to revise the lab Java Interface type that was presented during the programming course.
  • The lab concludes with an exercise to test your knowledge of the subject.
  • Accompanying the lab will be a short associated presentation.
